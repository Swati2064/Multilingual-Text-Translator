# 🌍 Language Translation App

A simple and interactive **Language Translation Web Application** built using **Python** and **Streamlit**. The application allows users to translate text into multiple languages and listen to the translated text using Text-to-Speech (TTS).

---

## 🚀 Features

- 🌐 Translate text into multiple languages
- 🎤 Convert translated text into speech
- 🔊 Play translated audio directly in the application
- 📥 Download the generated audio (.mp3)
- 🎨 Simple and user-friendly Streamlit interface
- 🌍 Supports dozens of world languages

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- mtranslate
- gTTS (Google Text-to-Speech)
- Base64

## 📦 Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Language-Translation-App.git
```

### Navigate to the Project Folder

```bash
cd Language-Translation-App
```

### Install Required Libraries

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
